# AB MALAYA - Premium Corporate Website Redesign

A high-end, premium corporate website for **AB Malaya Sdn Bhd**, a leading contractor and technical service provider in the Maritime, Oil & Gas, and Energy sectors in Malaysia.

## 🚀 Overview
This project is a high-performance PHP-based redesign of the AB MALAYA corporate profile. It emphasizes a "Marine Authority" aesthetic with modern web standards, featuring glassmorphism, fluid animations, and a sophisticated industrial design.

## 🛠 Tech Stack
- **Core**: PHP 8.x (Clean Architecture)
- **Styling**: Vanilla CSS3 (Premium Glassmorphism & Custom Transitions)
- **Typography**: 
    - **Cinzel**: Used for Hero Headers (Premium & Authority)
    - **Outfit**: Used for Section Headers (Modern & Clean)
    - **Inter**: Used for Body Text (Readability)
- **Routing**: Custom `router.php` for Clean URLs (e.g., `/about` instead of `about.php`).

## 📂 Project Structure
- `index.php`: High-impact landing page with localized hero content.
- `about.php`: Company journey with Glassmorphism Vision & Mission cards.
- `services.php`: Division-based service catalog with elegant accordion functionality.
- `careers.php`: Professional job portal and recruitment information.
- `contact.php`: Integrated contact hub with official business location.
- `privacy.php`: Kebijakan Privasi (Data protection).
- `terms.php`: Ketentuan Layanan (Usage terms).
- `service-terms.php`: Term & Service (Operational agreements).
- `includes/`: Modular components (`header.php`, `footer.php`).
- `assets/`:
    - `css/style.css`: Core design system and global animations.
    - `fonts/`: Custom Cinzel and Relicta assets.
    - `img/`: High-resolution industrial assets (e.g., `kapal-tanker.jpg`).

## ✨ Key Features
- **Clean URLs**: Powered by `router.php`. No `.php` extensions in the browser address bar.
- **Premium Navigation**:
    - **Mobile Burger**: Auto-hiding menu with "X" close icon and backdrop click-to-close logic.
    - **Full-Width Header**: Spans the entire screen for a "limitless" professional feel.
- **Floating UI**:
    - **Right-Side Scroll Indicator**: Elegant vertical line that tracks progress dynamically.
    - **Scroll-to-Top**: Minimalist floating button for easy navigation.
- **Content-Ready**: All text extracted and preserved in `ABMALAYA_Content_Clean.txt` for future updates.

## 🖥 Local Development
To run this project locally with Clean URLs, use the built-in PHP server:
```bash
php -S localhost:8000 router.php
```
Access the site at `http://localhost:8000`.
